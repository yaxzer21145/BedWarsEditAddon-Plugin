# BedWarsYA - BedWars1058 Integration Plugin

BedWarsYA 是一个基于 BedWars1058 插件的深度整合插件，为 BedWars 游戏添加了多种新功能和改进。

## 功能特性

### 1. 资源转换为经验系统
- 将铁锭、金锭、绿宝石转换为对应的经验值
- 可在 `XpBw.yml` 中配置每种资源的经验值
- 可配置哪些类型的竞技场开启经验转换

### 2. 加血事件系统
- 游戏开始后定时为所有玩家加血
- 支持1-2次加血，可自定义加血时间
- 加血时显示标题、副标题和聊天栏信息
- 可在 `HealthAdd.yml` 中修改加血时间和信息

### 3. 新增道具系统
- **自救平台**：右键烈焰棒生成粘液块平台，有冷却时间
- **降落伞**：右键羽毛迅速向上飞行
- **魔法羊毛**：右键羊毛生成延伸路径
- **回城卷轴**：右键火药开始回城倒计时，可移动取消

### 4. 搭桥蛋修改
- 改进了搭桥蛋的生成逻辑
- 支持水平和垂直投掷
- 生成更宽的桥面（2格宽度）

### 5. 战绩系统
- 完整的战力等级系统
- 五个阶梯：坚韧黑铁、秩序白银、荣耀黄金、闪耀钻石、至尊合金
- 每个阶梯包含多个等级
- 击杀、拆床、最终击杀都会增加战力
- 升级时显示标题提示
- 支持多种变量显示战力信息

### 6. 变量系统
- 支持多种变量用于消息和显示
- 队伍人数、游戏时间、战力信息等
- 可与PlaceholderAPI集成

### 7. 双语支持
- 支持中文和英文两种语言
- 玩家可自由切换语言
- 可扩展支持更多语言

### 8. BedWars1058深度整合
- 获取队伍出生点信息
- 获取队伍人数信息
- 监听游戏内事件（击杀、拆床等）
- 自动增加战力

## 安装方法

1. 确保服务器已安装 BedWars1058 插件
2. 将 `BedWarsYA.jar` 文件放入服务器的 `plugins` 文件夹
3. 启动服务器，插件会自动生成配置文件
4. 根据需要修改各配置文件中的参数
5. 重启服务器使配置生效

## 配置文件

插件生成以下配置文件：

- `Config.yml` - 主配置文件，控制各功能开关
- `XpBw.yml` - 资源转换为经验的配置
- `HealthAdd.yml` - 加血事件的配置
- `Shop.yml` - 商店和道具的配置
- `message.yml` - 中文消息配置
- `message_en.yml` - 英文消息配置
- `shop_items.yml` - 商店物品显示配置

## 命令

- `/bwya` - 查看帮助信息
- `/bwya lang` - 查看当前语言
- `/bwya lang <语言>` - 切换语言 (zh_CN 或 en_US)
